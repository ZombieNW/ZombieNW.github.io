About

Has this ever happened to you? You want to open a rom from file explorer, but emulators want a bajillion cli parameters? Well, ArchOpen is here for you! It's a simple program that allows you to open roms from file explorer by mapping a list of rom file extensions to their respective emulator cores.

Setup Instructions

  • Download ArchOpen below.
  • Extract the zip file.
  • Put the program in a safe place.
  • Create a config file with archopen.exe --generate-config.
  • Customize the config file to your liking.
  • Right click a rom file on your computer from explorer.
  • Select "Open with" and then "Choose another app".
  • Navigate to the directory where you extracted the zip file and select archopen.exe.
  • Click "OK" to save the file association.
