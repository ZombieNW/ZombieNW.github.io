The What?

Easily set up file associations for your rom collection. For those who don't like frontends or just want to easily open a rom sent to them.

Usage:
Grab the latest release below, and generate a config by running "archopen.exe --generateconfig".
Customize your config to your liking so each extension cooresponds with a core of your chosing.
Then just set the file type to always open with archopen.exe
