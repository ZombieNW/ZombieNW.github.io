About

I am the "Forever DM" of Minecraft Server Admins. I've been server admin-ing for like 8 years. Time and time again I've had to update someone else's mod or plugin because players want to play on a specific version of the game. This time, I've decided to take matters into my own hadns by making the plugin myself. This way, I can update it when the new version comes out and it looks like responsibility.

Setup Instructions

  • Download TradeShop for your server version.
  • Put .jar in /plugins folder.
  • Restart server.
  • Place sign on a container block (chest, barrel, shulker, etc).
  • Type out following format 
    [Shop] 
    X ItemName
    for 
    Y ItemName
  • Note! ItemName must be a valid Bukkit Material. Case/Space doesn't matter.
  • Alternatively! 
    [Shop] 
    X [Hand]
    for 
    Y [Hand]
    Use [Hand] as a placeholder; then right click the sign with the items.
  • If successful, the sign will reformat itself.
  • Right click the sign to trade !
